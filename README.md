Dial-Timer-Web
===============

A small touch & mouse-friendly circular dial timer built as a static, backendless site (GitHub Pages friendly).

Features
- Circular dial you can drag to set time
- Start / Pause / Reset controls
- Quick presets (5, 10, 15, 25 minutes)
- Touch & pointer friendly
- Lightweight (HTML/CSS/JS only)

Publish
1. Create a repo on GitHub and push this folder.
2. Enable GitHub Pages on the repository (branch: main or gh-pages) or use `gh` CLI.
3. (Optional) Add a CNAME to set a custom domain.

Suggested domain: dialtimer.app  (register separately)
